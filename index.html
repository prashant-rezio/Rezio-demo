Demo Instructions for Rezio AI Assistant

  1. Interact with Both Chats: Use the chat windows below to interact as both the buyer and the agent simultaneously.
  2. Login Credentials: When prompted, use the following credentials:
    • Buyer: Username - yc-buyer, Password - buyer123
    • Agent: Username - yc-agent, Password - agent123
  3. Start the Chat: As the buyer, click on the FIRST property listed at the top to start the chat.
  4. Agent Chat: In the agent's chat, click on the button located right under "Ongoing Negotiations."
  5. Ask Questions and Make Offers: As a buyer, feel free to ask questions about the property and make offers.
  6. Submitting Documents: Whenever you are requested for a cheque or a document, simply say, "Here is the cheque," "Here are the documents," or enter a number if an ID is requested.
  7. Refreshing the Chat: If you need to refresh the chat, go to the agent's side and message, "Refresh this chat," then log in again.
  8. Agent Intervention: If a question requires the agent's intervention, check the agent's chat for what the AI is asking for, and respond on behalf of the agent.
